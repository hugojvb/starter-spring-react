# Starter for Spring with React

Made for a quick start in full stack projects using Spring and React.

### Installation:

1. Clone the project

2. Run **npm i** in root directory

### Backend Additional Libraries:

-   Spring Boot
-   Spring Data Jpa
-   Spring Security
-   Lombok

### Frontend Additional Libraries:

-   Typescript
-   Material-ui
-   Material-ui icons
-   React-hook-form
-   React-query
-   React-router-dom
-   React-window
-   React-redux
-   Redux
-   Axios
